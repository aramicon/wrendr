Wrendr Slide Viewer

Welcome to the slides Viewer. This viewer will load slides text from a simple text file and allow you to move through these slides on display windows manually.

Instructions

  • Select a text file containing your slides using the "Choose slides File" button
  • Open a display window
  • Double-Click on a slide to display it
  • Move through the text slides using the NEXT and PREVIOUS arrows
